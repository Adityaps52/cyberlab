Introduction to Programming

Language is a medium of communication. People can interact with each other if they know a common language. Similarly, a user can interact with computer only if he/she knows a computer language. This computer language is called as programming language .

A language that is used to communicate with the computer system is termed as a programming language.

Computer programming languages are broadly classfied into two types i.e low-level language and high-level language.

The languages that are directly understood by the computer without any translator program is called as Low Level Programming language whereas the language that is understood by the programmer is termed as the high level language .

Let us understand the types of computer programming language in details.

To be more precise computer programming language can be categorized into 3 types i.e low level, middle level and high level.

  1. Low Level Language
  2. Middle Level Language
  3. High Level Language

Low Level Language:

  • The language that is directly understood by the computer system without translator program is called as low level programming language.
  • The low level language generally called as the Binary Lanaguge which contains only two digit 0 and 1 where 0 stands for "OFF" and 1 stands for "ON" .
  • These languages are designed to be close to the machine code or binary instructions understood by the computer's processor. Low-level languages are typically used to write programs that directly control the computer hardware or to perform tasks that require precise control over system resources.
  • Programs written in low-level languages can directly access memory addresses, registers, and other hardware resources. While low-level languages provide a high degree of control and efficiency, they are often more difficult to learn and use compared to high-level languages, which offer greater abstraction and ease of programming.

High level Language

  • The language that is familier to programmer or user is termed as high level language. This language used the english literature to instruct the computer.
  • A high-level language is a programming language that is designed to be easier for humans to read, write, and understand compared to low-level languages. These languages provide a higher level of abstraction from the underlying hardware and are closer to natural human language.
  • Overall, high-level languages offer programmers increased productivity, readability, and portability, at the expense of some performance and low-level control compared to low-level languages.
  • Some of the example of High level programming languages are: Java, Python, C, C++ and many more

Translator is required high level programming language as all the instruction is written in english language and computer only understand the machine level / Binary language
