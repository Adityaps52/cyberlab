Functions in Excel

  1. SUM():

    2. This Function is used to Calculate the sum of the values present in the cell.

    =SUM(A6:B6)


  2. AVERAGE():

    3. The AVERAGE function is a premade function in Excel,which calculates the average(airthmetic mean).


    =AVERAGE(B6:E6)


  3. MAX():

    4. This formula find the maximum value in the range(Cell Refernce/Cell Range).

    =MAX(J4:J19)


  4. MIN():

    5. This formula finds the minimum value in the range E1 to E4.

    =MIN(A2:A7)


  5. IF():

    6. This formula checks if the value in cell F1 is greater than 10. If it is, it returns "Pass"; otherwise, it returns "Fail".

    =IF(F1>10, "Pass", "Fail")


  6. COUNT():

    7. This formula counts the number of cells that contain numerical values in the range D2 to G4.

    =COUNT(D2:G4)


  7. COUNTIF():

    8. This formula counts the number of cells in the range Q1 to Q10 that are greater than 50.

    =COUNTIF(Q1:Q10, ">50")


  8. IF():

    9. This formula checks if the value in cell F1 is greater than 10. If it is, it returns "Pass"; otherwise, it returns "Fail".

    =IF(F1>10, "Pass", "Fail")


  9. VLOOKUP():

    10. This formula looks for the value in cell G1 in the first column of the range A1 to B10 and returns the corresponding value from the second column.

    =VLOOKUP(G1, A1:B10, 2, FALSE)


  10. CONCATENATE():

    11. This formula combines the text "Hello" and "World" to create the string "Hello World".

    =CONCATENATE("Hello ", "World")


  11. TODAY():

    12. This formula returns the current date.

    =TODAY()


  12. LEN():

    13. This formula calculates the number of characters in the text contained in cell H1.

    =LEN(H1)


  13. TRIM():

    14. This formula removes excess spaces from the text in cell I1.

    =TRIM(I1)


  14. LEFT():

    15. This formula extracts the leftmost 5 characters from the text in cell J1.

    =LEFT(J1, 5)


  15. RIGHT():

    16. This formula extracts the rightmost 3 characters from the text in cell K1.

    RIGHT():


  16. MID():

    17. This formula extracts 5 characters starting from the 3rd character of the text in cell L1.

    =MID(L1, 3, 5)


  17. UPPER():

    18. This formula converts the text in cell M1 to uppercase.

    =UPPER(M1)


  18. LOWER():

    19. This formula converts the text in cell N1 to lowercase.

    =LOWER(N1)


  19. PROPER():

    20. This formula capitalizes the first letter of each word in the text in cell O1.

    =PROPER(O1)


  20. ROUND():

    21. This formula rounds the value in cell P1 to 2 decimal places.

    =ROUND(P1, 2)


  21. SUMIF():

    22. This formula sums the values in the range S1 to S10 where the corresponding cell in the range R1 to R10 is "Apples".

    =SUMIF(R1:R10, "Apples", S1:S10)


  22. AVERAGEIF():

    23. This formula calculates the average of the values in the range U1 to U10 where the corresponding cell in the range T1 to T10 is "Red".

    =AVERAGEIF(T1:T10, "Red", U1:U10)


  23. IFERROR():

    24. This formula divides the value in cell V1 by the value in cell W1 and returns "Error" if an error occurs.

    =IFERROR(V1/W1, "Error")


  24. INDEX():

    25. This formula returns the value in the 3rd row and 2nd column of the range X1 to Y10.

    =INDEX(X1:Y10, 3, 2)


  25. MATCH():

    26. This formula searches for the value in cell Z1 in the range AA1 to AA10 and returns its position.

    =MATCH(Z1, AA1:AA10, 0)


  26. INDIRECT():

    27. This formula creates a reference to a cell based on the value in cell AB1.

    =INDIRECT("A"&AB1)


  27. TEXT():

    28. This formula converts the date in cell AL1 to a specific text format.

    =TEXT(AL1, "dd-mmm-yyyy")


  28. NETWORKDAYS():

    29. This formula calculates the number of working days between the dates in cells AM1 and AM10, excluding weekends.

    =NETWORKDAYS(AM1, AM10)


  29. RAND():

    30. This formula generates a random decimal number between 0 and 1.

    =RAND()

  30. RANDBETWEEN():

    31. This formula generates a random whole number between 1 and 100.

    =RANDBETWEEN(1, 100)


  31. SUBTOTAL():

    32. This formula calculates the sum of the visible cells in the range AQ1 to AQ10, ignoring any hidden rows.

    =SUBTOTAL(1, AQ1:AQ10)


  32. TRANSPOSE():

    33. This formula transposes the values in the range AR1 to AT1, changing them from a row to a column or vice versa.

    =TRANSPOSE(AR1:AT1)


  33. NOW():

    34. This formula returns the current date and time.

    =NOW()


  34. DATE():

    35. This formula creates a date based on the specified year (2023), month (12), and day (31).

    =DATE(2023, 12, 31)


  35. TIME():

    36. This formula creates a time based on the specified hour (9), minute (30), and second (0).

    =TIME(9, 30, 0)


  36. WEEKDAY():

    37. This formula returns the day of the week corresponding to the date in cell AS1.

    =WEEKDAY(AS1)


  37. EOMONTH():

    38. This formula returns the last day of the month, 3 months after the date in cell AT1.

    =EOMONTH(AT1, 3)


  38. DATEDIF():

    39. This formula calculates the number of complete years between the dates in cells AU1 and AU10.

    =DATEDIF(AU1, AU10, "y")


  39. PTM():

    40. This formula calculates the monthly payment for a loan with an annual interest rate of 5%, 12 monthly payments per year, and a principal amount of $20,000.

    =PMT(0.05/12, 12*5, -20000)


  40. FV():

    41. This formula calculates the future value of an investment with an annual interest rate of 10%, 10 periods, a regular payment of $100, and an initial principal of $500.

    =FV(0.1, 10, -100, -500)


  41. PV():

    42. This formula calculates the present value of an investment with an annual discount rate of 8%, 5 periods, a regular payment of $100, a future value of $0, and payments made at the end of each period.

    =PV(0.08, 5, -100, 0, 1)


  42. NPV():

    43. This formula calculates the net present value of cash flows in cells A1 to A5, with a discount rate of 10%, and adds the initial investment in cell A1.

    =NPV(0.1, A1:A5) + A1


  43. IRR():

    44. This formula calculates the internal rate of return for a series of cash flows in cells A1 to A5.

    =IRR(A1:A5)


  44. CUMIPMT():

    45. This formula calculates the cumulative interest paid on a loan with an annual interest rate of 5%, 12 monthly payments per year, a principal amount of $20,000, and payments made from the first to the 36th month.

    =CUMIPMT(0.05/12, 12*5, -20000, 1, 12*3)


  45. CUMPRINC():

    46. This formula calculates the cumulative principal paid on a loan with an annual interest rate of 5%, 12 monthly payments per year, a principal amount of $20,000, and payments made from the first to the 36th month.

    =CUMPRINC(0.05/12, 12*5, -20000, 1, 12*3)


  46. PTM():

    47. This formula calculates the monthly payment for an annuity due, where payments are made at the beginning of each period.

    =PMT(0.05/12, 12*5, -20000, 0, 1)


  47. VDB():

    48. This formula calculates the depreciation of an asset for the first 3 periods using the declining balance method.

    =VDB(1000, 100, 5, 1, 3)


  48. HLOOKUP():

    49. This formula looks for the value "Apples" in the first row of the range AV1 to AX10 and returns the corresponding value from the second row.

    =HLOOKUP("Apples", AV1:AX10, 2, FALSE)


  49. INDEX-MATCH():

    50. This formula combines the INDEX and MATCH functions to find the row that contains "Oranges" in column AW and returns the value from the second column in the range AV1 to AX10.

    =INDEX(AV1:AX10, MATCH("Oranges", AW1:AW10, 0), 2)


  50. OFFSET():

    51. This formula returns a range of cells starting from cell AY1, moving 3 rows down and 2 columns to the right, with a height of 2 rows and a width of 1 column.

    =OFFSET(AY1, 3, 2, 2, 1)


  51. SUMPRODUCT():

    52. This formula multiplies the values in the range AZ1 to AZ5 by the values in the range BA1 to BA5 and returns the sum of the products.

    =SUMPRODUCT(AZ1:AZ5, BA1:BA5)


  52. RANK():

    53. This formula calculates the rank of the value in cell BB1 compared to the values in the range BB1 to BB10.

    =RANK(BB1, BB1:BB10)


  53. STDEV():

    54. This formula calculates the standard deviation of the values in the range BC1 to BC10, representing a sample.

    =STDEV(BC1:BC10)


  54. STDEVP():

    55. This formula calculates the standard deviation of the values in the range BD1 to BD10, representing the entire population.

    =STDEVP(BD1:BD10)


  55. MEDIAN():

    56. This formula calculates the median (middle value) of the values in the range BE1 to BE10.

    =MEDIAN(BE1:BE10)
