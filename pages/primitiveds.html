Primitive Data Structure

Primitive Data Structureare the fundamental data structures avaialable in a programming language and allow storing the values of only one data type. Primitive Data Strucutre are the built-in types provided by the language itself. They are directly supported by the hardware and have a specific memory allocation and operations associated with them.

The primitive data strucutre are predefined in a programming language. The Type and Size of these data structure are already defined. There 4 main primitive data structures named integer, float, character and boolean. They are also the most used primitive data structures. Other than that, some other data types are : long, byte,short and double.

Let us look at the primitive data types with their example

1. Integer:

TheInteger data type represents the whole number that can be positive, negative or zero. The integer data type is represented by "int" in programming language. There are different sized integer that can hold different range of values. An integer occupy 2bytes of computer memory.

  1. Short Int can store between the range -32768 to +32767
  2. Int can store between the range -2147483648 to +2147483647
  3. Long Int can store between the range -2147483648 to +2147483647

int age = 76;

2. Character:

TheCharacter data type that stores single character or say alphabet. Remember, character data structure can only hold one alphabet at one time. The character data structure is represented by "Char". One thing that should be kept in mind while using the character data type single quotation / double quotation is used. A character occupy 1 Byte of computer memory.

  1. Unsigned Char can store between the range 0 to 255
  2. Signed Char can store between the range -128 to 127

char value = 'C';

Float:

TheFloating data type is primitive data structure that store fractional data / decimal numbers. The floating data type is represented by "float" in programming language. The floating data type contains both an integer part as well as fractional part. The Floating data type can hold upto 7 decimal place.

float pie = 3.14;

4. Double:

TheDouble data type is primitive data structure that also store fractional data / decimal numbers. The Double data type is represented by "double" in programming language. The Double data type can hold upto 14 decimal place.

double pie = 3.145235412211;
