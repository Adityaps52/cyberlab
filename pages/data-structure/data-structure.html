Data Structure

Data Structureis a method of orgainizing and storing data in computer system so that it can be accessed and manipulated efficienly. The data structure is primarly used or designed to store data for the purpose of using the data with various alogrithms.

Algorithms is a step-by-step / sequence of instructions that is designed to solve a particular task.

The data structure provides a means of organizing and managing large amount of data effectievly, enabling faster operations and optimized memeory usuage. There are various types of data structures, each with its own characterstics and uses.let us discuss each of them in details.

Data Structure has been classified into two types based on their characterstics and uses. There are two types of data structure:i.e

  1. Primitive data structure
  2. Non-Primitive data Structure

Data Structure Plays an very crucial role in building sophisticated application. It is very important to choose proper data structure for each task. Choosing an ill suited data strucuture could result in slow runtimes or unresponsive code. To choose best data structure consider the following questions:

  1. Which kind of information will be stored?
  2. What is the best way to organize the data?
  3. Where should data presist or be kept after it is created?
  4. How will that information be used
  5. What aspects of memory and storage reservation management should be considered?

Data Types

Data type is a category of data that determines the type of value that a variable or an expression can hold in a programming language. Generally, the data type specifies the size, format and range of values that can be stored in a variable. The data type also defines the type of operatiion that can be performed on the data, the memory space required to store it, and the constraints on its usage. Some of the frequently used data types:

  1. Integer:Represents whole numbers without decimal point
  2. Floating:Represents real numbers with decimal point
  3. Character:Represents individual character / alphabets.
  4. Integer:Represents a sequence of characters or sentences.
  5. Boolean:Represents a logical value either true or false.

Beside to the above noted data types, we also have number of other data types,like array structure, union and many more.The data type also helps computer in allocating memory to the value received during the programming.
