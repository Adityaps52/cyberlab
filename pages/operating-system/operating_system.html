Operating System

Operating System is a system software that acts as interface between user and hardware. The operating system is also called as the master software which is responsible for managing other computer software's and hardware too. The operating system is the first software that get loaded into the RAM during the booting process. An Operating system is the low-level software that supports a computer's basic functions such as scheduling tasks and controlling computer's peripherals. The Operating system is also responsible for providing a suitable environment for other application software.
Some of the examples of the operating system are Windows, Linux, Unix, Ubantu and many more.

Architecture of Operating System

The operating system is intially loaded into the bare machine(hardware) which activates the hardware and generates a platform for other application software. Other application software are loaded on or above the operating system. The end user will interact on the application.

Function of Operating system

An operating system overally perform all the task and job. It is also responsible for allocation and deallocation of the resources. But, Some of the typical operating system functions may include Memory management, file management, process management, Device management and security. . Beside to the above, an operating system performs many more functions but the primary functions of Operating System.

  1. Memory Management
  2. Device Management
  3. Process Management
  4. File Management
  5. Security

Types of Operating System

The Operating System are categorised into 2 types on the basis of their interface i.e CUI and GUI.

  1. Command-line User Interface
  2. Graphical User Interface

Command-line User Interface

Command-line User Interface also known as Character user Interface is a method in which user can interact with the computer program. All the interaction are done through "Commands". The commands can be of one word or of more than a line of text. This type of interface is carried out purely with the help of "Keyboard". For performing any task, a command is to be written on the interface which gets executed and does the necessary operation.

  1. Advantages:
    • Commands are used instead of manually doing it.
    • Only one task is performed at a time and is efficient
    • Uses less memory as compared to GUI
    • Less expensive as command are not updately frequently.
  2. Disadvantages:
    • Commands must be memorised by the user in order to work on this interface
    • Only one task is done at a time.
    • The interface of CUI isless appealing and supports only the usage of keyboard

Graphical User Interface

A Graphical based user Interface operating system interface that uses icons, menus and a mouse to manage interaction with the system. All the interactions are carried through the pointing devices. The graphical user interfaces were created because command line interfaces were quite complicated and it was difficult to learn all the commands in it.

  • Advantages
    • The main advantages of GUI is its "Easy to use Environment"
    • GUI supports Multitasking which allows user to execute two or more programs at the same time.
    • GUI let user take advantages of shortcut keys to minimize strokes.
  • Disadvantages
    • GUI needs high amount of storage space
    • GUI are less flexible , only pre-programmed instructions can be executed
    • GUIs ar slower than CUI
